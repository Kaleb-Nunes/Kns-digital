## ⚡ Tecnologias & Engenharia (Stack V44)

O projeto foi construído sobre uma arquitetura **'Zero-Bloat'** para garantir carregamento instantâneo, SEO técnico impecável e máxima compatibilidade.

* **Frontend Core:** HTML5 Semântico + Tailwind CSS (CDN/JIT).
* **Interatividade:** Alpine.js (Lightweight JavaScript Framework).
* **Animações:** AOS (Animate On Scroll) + CSS Keyframes Customizados (Shimmer/Pulse/Glow).
* **Design System:** 'Dark Empire Edition' — Foco total em contraste, legibilidade e estética Cyberpunk/Enterprise.
* **Internacionalização (i18n):** Engine nativa em JS para troca instantânea entre **PT / EN / ES**.
